🎮 Об игре
BEER IMPERIUM — это эпический кликер, вдохновлённый лучшими играми жанра. Вы начинаете с одной кружки пива и прокачиваетесь до владельца гигантской пивной империи с сотнями заводов и триллионами монет!

🎯 Основная цель
Построить самую мощную пивную империю, обогнать всех игроков в топе и стать легендой пивного мира!

🎯 Как играть
🍺 Базовое управление
Тапайте по пиву в центре экрана — зарабатывайте монеты

Покупайте улучшения во вкладке "Магазин"

Стройте заводы во вкладке "Заводы" для пассивного дохода

Активируйте баффы для временного усиления

📈 Стратегия развития
🆕 Начало игры (0-1000 монет)
Тапайте активно, чтобы накопить первые 50 монет

Купите первое улучшение "Кружка" (+1 сила)

Продолжайте тапать и покупать улучшения

🏗️ Средняя стадия (1000-1M монет)
Купите несколько заводов для пассивного дохода

Активируйте баффы для быстрого фарма

Прокачивайте силу до 50-100

Копите на дорогие улучшения

👑 Поздняя стадия (1M+ монет)
Покупайте дорогие заводы (100к+)

Используйте супер-баффы для гигантских бустов

Соревнуйтесь в топе игроков

Становитесь легендой!
